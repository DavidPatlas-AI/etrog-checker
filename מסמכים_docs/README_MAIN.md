# 🍋 בודק אתרוגים - פרויקט מקיף

## 📋 סקירה כללית

פרויקט מתקדם לזיהוי זני אתרוגים ובדיקת הכשרות שלהם באמצעות טכנולוגיות AI ופיתוח מתקדמות.

## 🚀 תכונות עיקריות

### ✨ תצוגה פנימית מלאה
- **צפייה ישירה** - כל קובץ נפתח בתצוגה פנימית (modal) ללא צורך בפתיחת טאבים חדשים
- **תמיכה בכל סוגי הקבצים**:
  - 📄 **PDF** - תצוגה מלאה עם iframe
  - ⚛️ **קוד** (TSX, TS, JS, JSX, CSS, HTML, JSON, Python, Dart) - הדגשת תחביר עם highlight.js
  - 📝 **Markdown** - רינדור מלא עם marked.js
  - 📄 **טקסט רגיל** - תצוגה נקייה ומסודרת
- **טיפול בשגיאות** - הודעות ברורות אם קובץ לא נמצא או לא ניתן להציג
- **פעולות מתקדמות** - הורדה, פתיחה בטאב חדש, העתקת נתיב

### 🎛️ לוח בקרה מתקדם
- **חיפוש חכם** - חיפוש בקבצים, תיאורים ונתיבים
- **סינון מתקדם** - לפי סטטוס (פעיל, ניסיוני, ארכיון) וקטגוריה
- **סטטיסטיקות** - מידע מפורט על כל הגרסאות
- **ייצוא נתונים** - JSON ו-CSV

### 📁 מבנה מסודר
- **תיקיות מאורגנות** - כל גרסה בתיקייה משלה
- **ניווט קל** - ממשק אינטואיטיבי ונגיש
- **תיעוד מלא** - הסברים מפורטים לכל גרסה

## 🎯 איך להשתמש

### 1. התחלה מהירה
```bash
# פתח את index.html בדפדפן
# או הפעל שרת מקומי:
python -m http.server 8000
# ואז פתח: http://localhost:8000
```

### 2. צפייה בגרסאות
1. לחץ על "🚀 צפייה בגרסאות" בדף הבית
2. השתמש בחיפוש וסינון למציאת קבצים
3. לחץ על "👁️ צפייה" לכל קובץ לתצוגה פנימית
4. או לחץ על "🔗 פתח" לפתיחה בטאב חדש

### 3. תצוגה פנימית
- **קוד**: הדגשת תחביר אוטומטית
- **PDF**: תצוגה מלאה עם ניווט
- **Markdown**: רינדור מלא עם עיצוב
- **טקסט**: תצוגה נקייה ומסודרת

## 🛠️ פיתוח

### הוספת קבצים חדשים
1. הוסף את הקובץ לתיקייה המתאימה ב-`גרסאות_versions/`
2. עדכן את `versions-manager.js` עם פרטי הקובץ
3. הקובץ יופיע אוטומטית בכלי הצפייה

### התאמות עיצוב
- ערוך את `versions-styles.css` לעיצוב כללי
- ערוך את ה-CSS בדף הספציפי לעיצובים ייחודיים
