# 🚀 התחלה מהירה - בודק אתרוגים

## ✨ מה חדש?

### 🎉 תצוגה פנימית מלאה!
- **כל קובץ נפתח בתצוגה פנימית** - אין צורך בטאבים חדשים
- **תמיכה מלאה בכל סוגי הקבצים**:
  - 📄 **PDF** - תצוגה מלאה עם iframe
  - ⚛️ **קוד** - הדגשת תחביר אוטומטית (TSX, TS, JS, JSX, CSS, HTML, JSON, Python, Dart)
  - 📝 **Markdown** - רינדור מלא עם עיצוב
  - 📄 **טקסט** - תצוגה נקייה ומסודרת
- **פעולות מתקדמות**: הורדה, פתיחה בטאב חדש, העתקת נתיב
- **טיפול בשגיאות**: הודעות ברורות אם קובץ לא נמצא

## 🎯 איך להתחיל?

### שלב 1: פתיחת הפרויקט
```bash
# אפשרות 1: פתיחה ישירה
# פתח את index.html בדפדפן

# אפשרות 2: שרת מקומי (מומלץ)
python -m http.server 8000
# ואז פתח: http://localhost:8000
```

### שלב 2: ניווט ראשי
1. **דף הבית** (`index.html`) - נקודת הכניסה הראשית
2. **צפייה בגרסאות** - לוח בקרה מתקדם עם תצוגה פנימית
3. **מידע על הפרויקט** - הסברים מפורטים
4. **דף בדיקה** - בדיקת תכונות המערכת
