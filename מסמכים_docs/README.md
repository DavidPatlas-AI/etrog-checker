## 🎯 סקירה כללית

**בודק אתרוגים** היא מערכת מתקדמת לזיהוי זני אתרוגים ובדיקת הכשרות שלהם, הכוללת:

- 🔍 **זיהוי זנים מתקדם** - חזון איש, קלבריה, תימני, ישראלי
- 📖 **בדיקת כשרות הלכתית** - בלטלך, רסיסוס, פיטם, בשלות
- 📸 **צילום תלת-ממדי מקצועי** - מערכת צילום 360° עם הדרכה אינטראקטיבית
- 📊 **ניהול צילום מתקדם** - לוח בקרה מקצועי עם עיבוד קבוצתי
- 🧠 **ניתוח איכות AI** - דירוג אוטומטי וזיהוי פגמים
- 📚 **מחקר הלכתי מקיף** - מאמרים, מדריכים ותיעוד מקצועי

## 🚀 התחלה מהירה

### 1. הורדה והפעלה
```bash
# הורדת הפרויקט
git clone https://github.com/your-repo/etrog-checker.git
cd etrog-checker

# הפעלת שרת מקומי (מומלץ)
python -m http.server 8000
# או
npx serve .

# פתיחת הדפדפן
open http://localhost:8000
```

### 2. ניווט במערכת
- **🏠 דף הבית**: `index.html` - שער לכל הכלים
- **🔍 מנתח מתקדם**: `גרסאות_versions/claude/etrog-advanced-analyzer.html`
- **📸 צילום תלת-ממדי**: `גרסאות_versions/claude/3d-photography-guide.html`
- **📊 ניהול צילום**: `גרסאות_versions/claude/photo-management-dashboard.html`
- **📚 צפייה בגרסאות**: `צופה_גרסאות_versions-viewer/versions-dashboard.html`

## 🛠️ תכונות עיקריות

### 🔍 זיהוי זנים מתקדם
- **זיהוי חזון איש**: איכות הגבוהה ביותר
- **זיהוי קלבריה איטלקי**: איכות גבוהה מאוד
- **זיהוי תימני מובחר**: איכות גבוהה
- **זיהוי ישראלי מקומי**: איכות משתנה

### 📖 בדיקת כשרות הלכתית
- **בדיקת בלטלך**: נקודות שחורות = פסול
- **בדיקת רסיסוס**: נקודות חומות = כשר
- **בדיקת פיטם**: שלם או חסר לגמרי
- **בדיקת בשלות**: צבע אחיד וטבעי

### 📸 מערכת צילום תלת-ממדי מקצועית

#### מדריך צילום אינטראקטיבי
- **הדרכה ויזואלית**: סימון זוויות צילום בזמן אמת
- **מעקב התקדמות**: אחוז השלמה וזמן שחלף
- **הדרכה קולית**: הוראות מפורטות לכל שלב
- **בדיקת איכות**: וולידציה של כל תמונה

#### לוח ניהול צילום מקצועי
- **מעקב אתרוגים**: ניהול כל האתרוגים בצילום
- **עיבוד קבוצתי**: הוספת מספר אתרוגים בבת אחת
- **ייצוא נתונים**: CSV, JSON, דוחות מפורטים
- **בדיקת תקינות**: וולידציה אוטומטית של תמונות

#### כלי עזר לניהול קבצים
- **שמות קבצים אוטומטיים**: `ET001_angle_015.jpg`
- **מטא-דאטה מפורט**: ציוד, צלם, תאריך, הערות
- **פרוטוקול צילום**: 24 זוויות ב-360° + זוויות משלימות
- **עיבוד קבוצתי**: ניהול מאות אתרוגים במקביל

### 🧠 ניתוח איכות AI
- **דירוג צבע ויופי**: 1-100
- **בדיקת סימטריה**: מושלם/טוב/בינוני
- **ניתוח מרקם**: חלק/מחוספס/מבריק
- **הערכת ערך מסחרי**: מחיר משוער
