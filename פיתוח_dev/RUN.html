איך להריץ את אפליקציית React (תיקיית dev)

להרצה מקומית של האפליקציה המתקדמת (React + Vite):

  1. פתח חלון PowerShell בתיקייה הראשית של הפרויקט.
  2. הפעל את הפקודות הבאות:
cd פיתוח_dev
npm install
npm run dev

לאחר ההרצה הופיע קישור, למשל http://localhost:5173. פתח אותו בדפדפן.

אם זו הפעם הראשונה, ההתקנה עלולה לקחת מספר דקות. אם יש שגיאה, נסה npm cache clean --force ואז שוב npm install.
